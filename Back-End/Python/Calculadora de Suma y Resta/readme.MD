 Se trata de una función que recibe una lista de strings con problemas aritmeticos simples (suma y resta) y devuelve los resultados de dichso problemas de forma vertical, uno
al lado del otro. El usuario tiene la libertad de decidir si quiere recibir el resultado o si simplemente quiere ver en esta forma más comoda la operación que está haciendo. 
Veamos un ejemplo:
"Función recibida:
arithmetic_arranger(["32 + 698", "3801 - 2", "45 + 43", "123 + 49"])
Devuelve:







   32      3801      45      123
+ 698    -    2    + 43    +  49
-----    ------    ----    -----
"
