# Demostar-Conocimientos
 El proyecto va a estar divido en carpetas. El nombre de las mismas dice el lenguaje utilizado/s sin incluir ninguno más. La idea es demostrar todo lo que puedo hacer solo usando los lenguajes utilizados (aunque puede quedar algo fuera que lo use en otra carpeta, donde lo uso con otros lenguajes).
  Las carpetas "Solo HTML" y "HTML + CSS", aunque no terminadas, comenzaron antes de que sean traídas a GitHub. Por ello, son los únicos cuyos primeros cambios no aparecen.
